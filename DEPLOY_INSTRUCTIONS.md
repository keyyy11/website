## ğŸ“‹ Langkah Seterusnya: Enable GitHub Pages

### **Step 1: Buka GitHub Repository**
1. Pergi ke: **https://github.com/keyyy11/website**
2. Pastikan anda sudah login ke GitHub account

### **Step 2: Buka Settings**
1. Klik tab **Settings** di bahagian atas repository
   - (Settings adalah tab terakhir di bahagian atas, sebelah kanan)

### **Step 3: Pergi ke Pages Section**
1. Di sidebar kiri, scroll down dan cari **Pages**
2. Atau terus ke: **https://github.com/keyyy11/website/settings/pages**

### **Step 4: Configure GitHub Pages**
1. Di bahagian **Source**, anda akan lihat dropdown
2. Pilih:
   - **Branch**: `main` (atau `master` jika ada)
   - **Folder**: `/ (root)` - pilih root folder
3. Klik butang **Save**

### **Step 5: Tunggu Deployment**
- GitHub akan process deployment anda
- Biasanya mengambil **1-2 minit**
- Anda akan lihat notification atau check di tab **Actions**

### **Step 6: Akses Website Anda**
- Selepas deployment siap, website anda akan available di:
- **URL**: `https://keyyy11.github.io/website/`
- Atau: `https://keyyy11.github.io/website/index.html`
